# AI Spend Auditor

AI Spend Auditor is a web application designed for startup founders and engineering teams to analyze their AI tool expenses and identify potential cost savings. It provides actionable recommendations based on tool usage, helping teams avoid overspending.

---

## 🚀 Features

- Input AI tools, plans, and number of users  
- Get instant audit results with:
  - Monthly savings
  - Yearly savings  
  - Plan recommendations  
  - Clear reasoning  
- Simple and clean UI  
- Basic audit logic for real-world scenarios  

---

## 🧠 How It Works

1. User selects:
   - Tool (ChatGPT, Copilot, etc.)
   - Plan (Plus, Team, Business)
   - Number of users  

2. The audit engine:
   - Evaluates plan efficiency  
   - Detects overpayment cases  
   - Suggests better alternatives  

3. Output:
   - Total savings  
   - Recommendation  
   - Reason for suggestion
